Site Configuration

All site-level settings live in site.config.js at the project root. The file is a plain CommonJS module — no build step required. Both the ingest pipeline and the Next.js/Nextra layer read from it at their respective run times.

Fields

FieldTypeDefaultDescription
titlestring—Site name shown in browser tab and navbar logo
base_urlstring—Full origin URL (e.g. https://user.github.io) used in sitemap/SEO
base_pathstring''Sub-path prefix (e.g. /repo-name) for GitHub Pages project sites
repo_urlstring''GitHub repo URL; shows circular GitHub icon in navbar when non-empty
theme_toggle'navbar'|'sidebar''sidebar'Where the light/dark/system toggle appears
ingest_readmebooleanfalseCopy README.md → docs/about.md before ingesting
feedbooleantrueEnable inline page continuation below each page
max_feed_pagesnumber20Max continuation sections before showing a "Continue →" link
tocbooleantrueShow right-side "On This Page" TOC column
meta_sidebarbooleantrueShow metadata sidebar below TOC
nav_orderobject{}Ordered slug arrays keyed by directory path ('' = root)

Requirements

  1. REQ-1: base_path is prepended to all redirect URLs generated by auto_index() and all feed continuation links
  2. REQ-2: When feed: false, PageContinuation renders nothing and feed-index.json is not written
  3. REQ-3: When repo_url is empty, the GitHub icon is hidden from the navbar
  4. REQ-4: When theme_toggle: 'navbar', the toggle appears in the navbar and Nextra's built-in dark mode toggle is hidden
  5. REQ-5: nav_order arrays set explicit page order for the given directory; unlisted slugs sort alphabetically after ordered entries
  6. REQ-6: max_feed_pages limits continuation sections; the first page beyond the limit is shown as a "Continue →" link target

Test Cases

REQ-1 is covered by tests/build/ingest.test.js (test_auto_index_targets_first_sorted_page).

REQ-5 and REQ-6 are covered by tests/build/ingest.test.js (sort_entries suite) and tests/components/PageContinuation.test.jsx (test_continuation_pagination_link).

REQ-2, REQ-3, REQ-4 are verified manually:

  • Set feed: false → no feed sections rendered, feed-index.json not written
  • Set repo_url: '' → GitHub icon absent from navbar
  • Set theme_toggle: 'navbar' → toggle appears in navbar, sidebar toggle absent
